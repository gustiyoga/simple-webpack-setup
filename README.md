# Simple Webpack Setup
A simple Webpack 4 setup

## How to use
First you need to install dependencies

```
npm install
```

and go build with

```
npm run build
```

## Built With
* [Webpack](https://github.com/webpack/webpack)

### Dependencies
* autoprefixer
* babel-core
* babel-loader
* babel-preset-env
* clean-webpack-plugin
* copy-webpack-plugin
* css-loader
* mini-css-extract-plugin
* node-sass
* optimize-css-assets-webpack-plugin
* postcss-loader
* sass-loader
* webpack
* webpack-cli
* webpack-merge
